## 📌 Project Overview

AI Resume Analyzer is a modern full-stack SaaS-style web application that helps users analyze resumes using ATS-based scoring and AI-generated insights.

The platform allows users to upload resumes, receive intelligent feedback, detect missing keywords, and improve resume quality for better recruiter visibility.

# ⚙️ Local Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/jeba57/Resume-Analyzer.git
```


---


## 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```


---


## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```


---


# 🔑 Environment Variables

Create `.env` inside backend:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=your_groq_api_key
```
